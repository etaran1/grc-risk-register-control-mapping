# Policy Excerpt: Access Control (Sample)

## Purpose
Ensure access is limited to authorized users and reviewed regularly.

## Requirements
1. MFA is required for all remote access.
2. Privileged accounts must use separate admin identities.
3. Access to critical systems requires least privilege.
4. Privileged access must be reviewed quarterly with documented evidence.

## Evidence
- MFA enforcement report
- Quarterly access review attestation and reviewer signatures
- Privileged group membership export
